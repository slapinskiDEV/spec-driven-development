# Plans

All planning documents are organized here by lifecycle stage.

## Folder Structure

- **[draft/](./draft/)** — Newly created plans, not yet being executed.
- **[in-progress/](./in-progress/)** — Plans that are actively being worked on (partially completed, blocked, or awaiting user input).
- **[archived/](./archived/)** — Completed or retired plans (moved here only on explicit user instruction).

## Templates

The [plan template](./template.md) is available for creating new plans.

See [`../README.md`](../README.md) and [`../agent.md`](../agent.md) for the full plan lifecycle rules.
